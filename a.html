인사 소개

안녕하세요? 조르디입니다!

저는 훌륭하고 멋진 소프트웨어 개발을 즐겨하는 사람입니다. 웹사이트 및 브라우저 플러그인, 기업 소프트웨어도 만들었고, 게임도 제작했습니다. 그 중 좋아했던 프로젝트 몇 개를 아래 포트폴리오로 만들었으니 여기를 누르시거나 페이지 아래쪽에서 구경해 보세요! 도움이 필요하세요? 저하고 이야기해 보시겠어요? 소셜 미디어나 이메일로 연락해 주세요.

개인 관심사

프로그래밍을 하지 않을 때는 다음과 같은 것으로 소일하며 즐거운 생활을 하고 있답니다.

  • 한국어 공부하는 것이 제일 즐거워요.
  • 좋아하는 e스포츠 경기를 봐요.
  • 전 세계 컨퍼런스에서 영감을 받고 있어요.
  • 그리고, 더 많은 사람들 만나 새로운 기회를 창출하고 있어요.

전문 기술

저는 여러 분야의 프로그래밍 언어를 다루어 왔습니다. 웹으로는 PHP와 자바를 사용합니다. 높은 성과율을 필요로 하는 것은 당연히 C++를 사용하고, 시제품 및 견본을 만들거나 취미용 프로젝트를 할 때는 Python 대신 쓰기도 합니다.

경력

데브옵스/소프트웨어 엔지니어

빅 데이터 • 데이터 분석 • 백엔드 개발
전임 소프트웨어 엔지니어로 종사하고 있는 현 회사에서는 처음에 시간제 직원으로 시작했습니다. 그 후 전임 인턴으로의 승진을 거쳐 현 데이터 부서의 자리까지 오게 되었는데요, 인터넷에서의 고품질 데이터 추출을 책임지고 습니다. 비구조적 딥웹(deep web)에서 고품질 데이터를 추출하고 그 추출 결과를 구조화된 데이터로 바꾸는 소프트웨어를 개발함으로써 분석이 가능하도록 하는 것이 제 담당 분야입니다.
회사: 쿠스토
기간: 2011년 11월 ~ 현재

프리랜서 웹 개발자

웹 개발 • 사용자 인터페이스 디자인 • 맞춤형 소프트웨어
오래 계획했던 개인 사업을 18살에 드디어 ‘카이와 프로젝트’라는 이름으로 시작했습니다. 카이와 프로젝트를 통해 여러가지 프리랜서 프로젝트를 많이 했습니다. 가장 많이 해 온 프로젝트로는 주로 맞춤형 솔루션으로, 고객관리 시스템, 인터넷 호텔과 게스트하우스 예약 시스템 및 예상 고객명단 작성(lead generation) 애플리케이션 등이 있습니다.
회사: 카이와 프로젝트
기간: 2009년 11월 ~ 현재

인턴 프로그래머

실행 계획 • C# 이용 개발 • 프로세스 최적화
바우하우스에서 제 졸업 인턴쉽을 했습니다. 인턴쉽동안 애플리케이션 3개 개발했습니다. 실행 계획 웹 인터페이스하고 프로세스 최적화하는 소프트웨어를 만들었습니다. 실행 계획 웹 인터페이스는 개선한 출입하는 배달 계획위에서 만들었고 프로세스 최적화하는 소프트웨어는 산업 공정 그림에서 예비 부품을 다 수집하고 리스트를 인화하기 전 자동화한 시스템을 개발했습니다. 이 시스템은 작업 부하가 이틀부터 마우스를 한번 클릭으로 감소했습니다.
회사: 바우하우스
기간: 2010년 2월 ~ 2010년 8월

교육

NHTV 브레다 과학대학교

컴퓨터 공학 학사 • 게임 개발
NHTV 브레다 과학대학교에서 국제 게임 아키텍처와 디자인 코스를 공부했습니다. 광범위한 분야에서 견고한 지식을 단단하게 쌓았습니다. 데이터 스트럭쳐, 컴퓨터 네트워킹 및 오디오와 그래픽스 프로그램이에 대해 특히 더 많이 배웠습니다.

2016년 초 예정

ROC Twente

전공: 컴퓨터 공학 • 소프트웨어 개발
ROC Twente에서 2개의 전공 과정을 졸업했습니다. 첫 과정은 컴퓨터 공학이었습니다. 이 과정을 끝낸 후 애플리케이션 개발 과정도 추가하여 끝냈습니다. 이 두 과정을 모두 들으면서 하드웨어와 소프트웨어를 둘 다 잘 이해할 수 있게 되었습니다. 원래 애플리케이션 개발 코스는 2년 과정이지만 저는 1년 일찍 조기졸업을 했습니다.
